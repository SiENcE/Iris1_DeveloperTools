This Application is User Interface Design Tool for Iris (UO) 3D-Client.

0.40:	-GNU Text (PO) Language support added for Gumps
	-CSL Code Editor added
	-Iris Gump testfunction added
	-Import of Templates is added

0.32:	-Iris Fonts are now inlcuded. This should fix the Textbox Error. Fonts are specified in config.xml.

GFM is an XML-FileFormat.
GFM loader code is iris/CSLForm.cpp.

Verdata.mul is loaded if available in UO Directory.


Please edit config.xml for UO Path before starting!!
