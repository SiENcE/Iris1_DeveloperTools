This Application is a Granny 1.0 (*.grn) Viewer for Ultima Online 3D Character/Monster/Clothes Models.

To run Grannyviewer:

Edit the xml/config.xml and setup the Ultima Online path !

----------------------------------------------------------

If GrannyViewer crashed, run immediate.bat.
Do not copy granny_aos.xml to iris/granny_aos.xml. It cause some problems.

Grannyviewer also allows to load SMD Files.

SMD model restriction:
 - only support 1 texture. (bmp not support)
 - matching model & animation bone name, not order.
 - root bone name is master.
 - weapon attatchment bone name is CP_Grasp_Lhand & CP_Grasp_Rhand.
 - compare edited models vs original models(hlsdk2.2_org/*)
